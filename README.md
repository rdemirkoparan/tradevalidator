# CodingTest3156
Trade Information Validator

## Environment
JDK 8
Embedded Tomcat 8


## To compile and run
1. git clone and go into directory
2. mvn compile
3. mvn package
4. java -jar target/tradevalidator-0.0.1-SNAPSHOT.jar
   
   
## Usage
1. run doc/client.html
